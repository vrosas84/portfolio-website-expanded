it415 final: part1

Posted by Vince Rosas on May 20, 2023

This project is my IT415 Final Project part 1. It was created in semester 20EW1 which ran from September through October of 2020. It is a project proposal written to convince decision makers of the benefit of the project.

Requirements and Knowledge

This project proposal required the ability to analyze computing and business problems and produce a solution. The proposal required analyzing the technical and non-technical risks associated with the project as well. The proposed solution required knowledge of advanced networking, a security-first approach, and project planning.

I chose this item because it displays multiple skills in one item. The private networking used in the project highlights knowledge of advanced networking. It also shows knowledge of security principles by building incorporating security protocols from the start instead of being an add-on. The detailed project timeline shows the ability to design a project and plan for unexpected events.

Key Takeaways

Most of what I learned from creating this artifact was how to think critically and examine problems from multiple angles. Creating this artifact required I examine the problem from a business perspective and a technical perspective. Creating this gives me useful experience in creating project proposals which will certainly be needed in my career as I advance.

Challenges

The biggest challenge in creating this artifact was time. Creating the artifact was not a problem but having the time to fully complete it was. I doubt this would be a problem in a work environment since I currently am splitting my time between work and school.
